UnliTyper ⌨️
UnliTyper is a lightweight, web-based typing practice application designed to help users improve their typing speed and accuracy. Whether you are a student, a professional, or just looking to improve your keyboard skills, UniTyper provides a clean, distraction-free environment to practice.

🚀 Features
Real-time Feedback: See your words per minute (WPM) and accuracy as you type.

Minimalist Design: A distraction-free interface built to keep you focused.

Responsive: Works seamlessly on desktops and tablets.

Fast Loading: Built with optimized code for a quick experience.

🛠️ Technologies Used
HTML5 & CSS3: For structure and styling.

JavaScript: For the typing logic, timer, and score calculation.

📦 How to Run Locally
If you want to run this project on your own computer:

Clone this repository:

git clone https://github.com/AntoAmalan/Unlityper.git

2. Open the `index.html` file in any modern web browser.

## 📝 Usage
*   Start typing the text displayed on the screen.
*   The timer begins as soon as you type your first character.
*   Review your WPM and accuracy once the session is complete.
